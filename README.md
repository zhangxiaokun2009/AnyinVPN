# AnyinVPN (安隐) 🎈

一个基于 WPF + C# 开发的简单代理客户端工具。

纯属个人闲暇无聊时写着玩的玩具项目，拿出来开源主要是留个纪念并供大家交流学习。

---

## 🌟 项目亮点与特点

* **🎨 界面 UI 还算看得过去**：
  * 使用了 WPF + Material Design 框架，做了无边框卡片化设计、圆角阴影以及自定义系统托盘，视觉效果和交互体验相对比较现代、干净。
* **📢 内置与跳转广告机制测试**：
  * **内置广告位**：基于 `WebView2` 集成了底栏嵌入式广告展示。
  * **弹窗/跳转广告**：测试了客户端常见的“广告变现”逻辑——在用户点击连接前弹出阻断窗口，并自动拉起外置浏览器（Microsoft Edge）进行广告页跳转及倒计时校验。
* **⚡ 核心底层对接**：
  * 解析 VMess / VLess / Trojan 等常见节点协议。
  * 通过 Windows 命名管道（Named Pipe）向 `sing-box` 内核动态传输配置，并调用 WinINet API 设置系统代理。

---

## 💣 吐槽与代码逻辑说明（吐槽警告）

先叠个甲：**前端 UI 确实花了点心思调了调，但后端的逻辑写得挺乱的！** 😅

* **代码架构**：典型的“能跑就行”风格，绝大部分逻辑都一股脑塞在了 `MainWindow.xaml.cs` 里，缺乏严谨的分层设计（MVVM 架构基本没用上）。
* **进程与网络管理**：对于进程杀除和错误处理比较粗暴，部分 URL 也是硬编码在代码里的。
* **给想跑起来玩的朋友**：
  如果你想下载本项目编译或者二次开发，**强烈建议你自己把底层逻辑重构和优化一下**（把硬编码的 API / 订阅地址替换成你自己的），不然可能会跑出各种意料之外的 Bug。

## 🛠️ 技术栈

* **GUI 框架**：WPF (.NET / C#)
* **UI 组件库**：[MaterialDesignInXamlToolkit](https://github.com/MaterialDesignInXAML/MaterialDesignInXamlToolkit)
* **系统托盘**：Hardcodet.Wpf.TaskbarNotification
* **Web 渲染**：Microsoft.Web.WebView2
* **内核引擎**：[sing-box](https://github.com/SagerNet/sing-box)
